# Secret Friend Draw

## Description

This project is a Secret Friend (Secret Santa) application developed with JavaScript. It allows users to register participants, perform random pairings, and display the draw results dynamically on the page.

The application uses JavaScript array manipulation, randomization techniques, and DOM updates to create an interactive and user-friendly experience.

## Features

### Participant Registration

* Add participants to the draw.
* Validate empty inputs.
* Display the list of registered participants.

### Random Pair Generation

* Randomly selects participants.
* Creates unique pairs for the draw.
* Displays the generated matches.

### Input Validation

* Prevents empty names from being added.
* Ensures at least two participants are available before drawing.

### Dynamic Interface Updates

* Updates participant lists in real time.
* Displays draw results instantly.

### Reset Functionality

* Clears participant lists.
* Clears draw results.
* Allows a new draw to be created.

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/GustavoPGCordeiro/secret-friend-draw.git
```

2. Open the project folder.

3. Open the `index.html` file in your browser.

4. Add participant names and start the draw.

## Core Functionalities

### Add Participants

* Enter a participant's name.
* Store the name in an array.
* Display all registered participants.

### Generate Random Pairs

* Randomly selects participants.
* Creates friend-to-friend assignments.
* Displays the results on screen.

### Restart the Draw

* Clears all current data.
* Allows a new participant list to be created.
